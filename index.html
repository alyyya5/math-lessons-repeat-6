Положительные и отрицательные числа · Модуль · Сравнение

📖 Теория

1. Координатная прямая

Прямая, на которой выбраны:

  • начало отсчёта (точка O, число 0),
  • положительное направление (обычно вправо или вверх),
  • единичный отрезок,

называется координатной прямой.
Числа справа от нуля – положительные (со знаком + или без знака).
Числа слева от нуля – отрицательные (со знаком –).

🔹 Пример: +5, 7, 0.3 – положительные; –2, –0.7, –3/4 – отрицательные.

2. Противоположные числа

Числа, отличающиеся только знаком, называются противоположными.
Например: 7 и –7. Ноль противоположен сам себе.
На координатной прямой они расположены симметрично относительно нуля.

📌 Если число a, то противоположное ему –a.

3. Модуль числа

Модуль числа n – это расстояние от начала отсчёта до точки N(n).
Обозначение: |n|. Модуль всегда неотрицателен.

|n| = n, если n ≥ 0   |   |n| = –n, если n < 0
📐 |–7| = 7,   |3,5| = 3,5,   |0| = 0.

Свойство: модули противоположных чисел равны: |–a| = |a|.

4. Сравнение чисел

  • Любое положительное число больше любого отрицательного.
  • Ноль больше отрицательного, но меньше положительного.
  • Из двух отрицательных чисел больше то, у которого модуль меньше.
  • На координатной прямой большее число находится правее.
✅ Пример: –5 < –3 (так как |–5|=5, |–3|=3, 5>3 → –5 < –3).
✅ 7 > –10,   0 > –2,5,   –1,2 < –0,8.
