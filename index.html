🧮 Занятие 1

Положительные и отрицательные числа · Модуль · Сравнение

📖 Теория

1. Координатная прямая

Прямая, на которой выбраны начало отсчёта (0), положительное направление и единичный отрезок – координатная прямая.
Числа справа от нуля – положительные, слева – отрицательные.

🔹 Пример: +5, 7, 0.3 – положительные; –2, –0.7, –3/4 – отрицательные.

2. Противоположные числа

Числа, отличающиеся только знаком: 7 и –7. Ноль противоположен сам себе.

3. Модуль числа

Модуль – расстояние от начала отсчёта до точки. Обозначение |n|. Всегда неотрицателен.

|n| = n, если n ≥ 0; |n| = –n, если n < 0
📐 |–7| = 7, |3,5| = 3.5, |0| = 0

4. Сравнение чисел

  • Положительное > любого отрицательного
  • Ноль > отрицательного, < положительного
  • Из двух отрицательных больше то, у которого модуль меньше
✅ –5 < –3 (|–5|=5 > |–3|=3 → –5 < –3)

➕ Занятие 2

Сложение и вычитание положительных и отрицательных чисел

📖 Теория

Сложение чисел с помощью координатной прямой

Прибавить положительное число – перемещение вправо, отрицательное – влево.

Правила сложения

• Сумма двух отрицательных: сложить модули, поставить «–».
–3 + (–5) = –(3+5) = –8
• Сумма чисел с разными знаками: из большего модуля вычесть меньший, поставить знак числа с большим модулем.
–7 + 4 = –3, 8 + (–5) = 3

Вычитание

a – b = a + (–b)
–5 – (–3) = –5 + 3 = –2
